 인터페이스
: 객체의 사용 방법을 정의한 타입. 개발 코드를 수정하지 않고, 사용하는 객체를 벼경할 수 있도록 하기 위함


 상수 필드 선언
: 인터페이스는 데이터를 저장할 수 없다.
: 대신 상수 필드만 선언할 수 있다. public static final
: 따라서 인터페이스에 선언된 필드는 모두 public static final 의 특성을 갖는다.

 추상 메소드 선언
: 인터페이스를 통해 호출된 메소드는 최종적으로 객체에서 실행된다. 그렇기 떄문에 인터페이스의 메소드는 추상 메소드로 선언한다.
: 인터페이스에 선언된 추상 메소드는 모두 public abstract 특성을 갖기 때문에 생략해도 된다.
